### NotePad-Android
<hr>

This app displays lists of Notes

Pre-requisites
--------------

- Android SDK v28
- Min. Android SDK 16

Features
---------------
- SQL Database
- Capability to Edit and Delete Notes
- Share Notes
- Search Notes 
- Adaptive Layout
- List Sorting
